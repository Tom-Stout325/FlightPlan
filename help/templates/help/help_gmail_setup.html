Setting Up a Gmail App Password

Use this guide to create a Gmail App Password—a special 16-character password that allows 12bytes software emails securely through Gmail.

Step 1: Turn on 2-Step Verification

  1. Go to myaccount.google.com/security.
  2. Under “How you sign in to Google”, click 2-Step Verification.
  3. Follow the prompts to enable it. Once enabled, return to the Security page.

Step 2: Open the App Passwords Page

After 2-Step Verification is on, click App passwords under the same Security section.

Step 3: Generate an App Password

  1. In Select app, choose Mail or select Other (Custom name) and type something like 12bytes.
  2. In Select device, choose your device or Other.
  3. Click Generate.
  4. Copy the 16-character password that appears (ignore spaces).

Troubleshooting

  • App passwords not showing? Make sure 2-Step Verification is turned on.
  • Advanced Protection user? App passwords are disabled for those accounts.
  • Google Workspace user? Your admin may need to enable app passwords.
  • Still can’t access? Wait a few minutes, sign out/in, or try a different browser.

Good Practices

  • Generate separate app passwords for each server or app.
  • Label them clearly for easy management.
  • Revoke old or unused app passwords regularly.
